QUICK START

1. Add routes to routes/api.php:

use App\Http\Controllers\Auth\{
    RegisterController,
    LoginController,
    LogoutController,
    PasswordResetController
};

Route::middleware('api')->group(function () {
    Route::post('/auth/register', RegisterController::class);
    Route::post('/auth/login', LoginController::class);
    Route::post('/auth/forgot-password', [PasswordResetController::class, 'sendReset']);
    Route::post('/auth/reset-password', [PasswordResetController::class, 'resetPassword']);

    Route::middleware('auth:sanctum')->group(function () {
        Route::post('/auth/logout', LogoutController::class);
    });
});

2. Run: php artisan serve

3. Test endpoint with cURL:
curl -X POST http://localhost:8000/api/auth/login \
  -H "Content-Type: application/json" \
  -d '{"email":"test@example.com","password":"TestPass123!"}'

================================================================================
WHAT EACH CONTROLLER DOES
================================================================================

RegisterController (AUTH-001)
- Accepts name, email, password, role
- Validates input (strong password, unique email)
- Hashes password
- Creates user with role
- Returns Sanctum token
- Status: 201 Created

LoginController (AUTH-002)
- Accepts email and password
- Validates credentials
- Returns Sanctum token on success
- Returns 401 Unauthorized for invalid credentials

LogoutController (AUTH-003)
- Deletes current access token
- Returns 200 OK

PasswordResetController (AUTH-004)
- sendReset(): Sends password reset email
- resetPassword(): Updates password with token validation
- Handles expired/invalid tokens with 422 error

API ENDPOINTS
================================================================================

POST /api/auth/register
  Input: {name, email, password, role}
  Output: {message, user, token}
  Status: 201

POST /api/auth/login
  Input: {email, password}
  Output: {message, user, token}
  Status: 200 or 401

  Status: 200

  Output: {message}
  Status: 200 or 422
